# P0-1 技術検証仕様

# 1. 概要

Rust + egui + eframe を利用した Runtime IDE 基盤の成立性を確認する。

# 2. 検証対象

- egui
- eframe
- egui_dock

# 3. 段階検証

P0-1 は以下順序で段階的に確認する。

## 3.1 最小 Window

最小 Window 表示のみ確認する。

### 確認内容

- eframe 起動
- Window 表示
- Event Loop

## 3.2 Docking

Window 表示成功後に Docking を確認する。

### 確認内容

- egui_dock
- Panel 分割
- Panel 移動

## 3.3 Layout Persistence

Dock Layout の保存・復元を確認する。

### 確認内容

- DockState serialize
- DockState deserialize
- Layout restore

## 3.4 Multi Panel

複数 Panel 表示を確認する。

## 3.5 GPU Viewport Placeholder

GPU Viewport 用 Panel を確認する。

# 4. Windows / Linux / macOS

## Windows

必須。

## Linux

Windows 成功後に確認する。

## macOS

実機未保有のため SKIP とする。

# 5. 今後の拡張

- wry
- WebView
- Runtime process
- Event / State
- Redux 型
