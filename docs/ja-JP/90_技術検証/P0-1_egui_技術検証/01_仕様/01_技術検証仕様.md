# P0-1 技術検証仕様

# 1. 概要

Rust + egui + eframe を利用した Runtime IDE 基盤の成立性を確認する。

# 2. 検証対象

- egui
- eframe
- egui_dock

# 3. 確認事項

## 3.1 Window

Window 表示が可能であること。

## 3.2 Docking

Docking UI が利用可能であること。

## 3.3 Multi Panel

複数 Panel を表示可能であること。

## 3.4 GPU Viewport

GPU Viewport 用 Panel を配置可能であること。

## 3.5 Event Loop

UI Event Loop が正常動作すること。

## 3.6 State 更新

State 更新後に UI 更新可能であること。

# 4. 今後の拡張

- wry
- WebView
- Runtime process
- Event / State
- Redux 型
