# Linux Native Window Title 検証ケース

# 1. locale

## TC-P0-1C-001

### 目的

locale が日本語設定か確認する。

### 手順

1. locale 実行
2. LANG 確認
3. LC_ALL 確認

### 期待結果

ja_JP.UTF-8 であること。

# 2. Wayland/X11

## TC-P0-1C-002

### 目的

Wayland/X11 差異を確認する。

### 手順

1. Wayland 実行
2. X11 実行
3. native title 比較

### 期待結果

差異有無を確認できること。

# 3. GTK

## TC-P0-1C-003

### 目的

GTK backend の影響を確認する。

### 手順

1. GTK 確認
2. native title 確認

### 期待結果

GTK 影響有無を確認できること。

# 4. eframe

## TC-P0-1C-004

### 目的

window title API の影響を確認する。

### 手順

1. ASCII title
2. 日本語 title
3. mixed title

### 期待結果

title 差異を確認できること。
