# P0-2 WebView 技術検証仕様

# 1. 概要

wry を利用した WebView と egui Runtime IDE の共存可能性を確認する。

# 2. 検証対象

- wry
- WebView
- HTML 表示
- URL 表示
- egui coexist

# 3. 段階検証

## 3.1 最小 WebView

最小 WebView 表示を確認する。

## 3.2 egui coexist

egui と WebView の同時利用を確認する。

## 3.3 Event 連携

WebView と IDE Event の連携可能性を確認する。

# 4. 想定用途

- Help System
- Documentation Viewer
- Browser Tool UI
- HTML Based Panel
- Scene 切替 UI

# 5. 非対象

- Runtime process
- AI Agent
- IPC
- GPU Viewport
