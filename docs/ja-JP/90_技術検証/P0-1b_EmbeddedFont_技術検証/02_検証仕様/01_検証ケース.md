# EmbeddedFont 検証ケース

# 1. Embedded Font

## TC-P0-1B-001

### 目的

Embedded Font により日本語表示可能か確認する。

### 手順

1. アプリ起動
2. 日本語 Panel 確認
3. 日本語ログ確認

### 期待結果

Linux / Windows で日本語表示可能であること。

# 2. cross-platform 一致

## TC-P0-1B-002

### 目的

OS 差異による文字化けが発生しないか確認する。

### 手順

1. Linux 実行
2. Windows 実行
3. UI 比較

### 期待結果

Linux / Windows 間で UI 差異が小さいこと。

# 3. Runtime Font Reload

## TC-P0-1B-003

### 目的

Runtime 中に font 切替可能か確認する。

### 手順

1. font 変更
2. UI 更新

### 期待結果

Runtime 中に font 更新可能であること。

# 4. Custom Font

## TC-P0-1B-004

### 目的

利用者 custom font を追加可能か確認する。

### 手順

1. custom font 配置
2. font 選択
3. UI 更新

### 期待結果

利用者 custom font を利用可能であること。
