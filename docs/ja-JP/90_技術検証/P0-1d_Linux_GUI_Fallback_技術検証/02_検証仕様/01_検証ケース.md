# Linux GUI Fallback 検証ケース

# 1. fallback script

## TC-P0-1D-001

### 目的

fallback script により Linux GUI が安定起動するか確認する。

### 手順

```bash
chmod +x ./scripts/run_linux_fallback.sh

./scripts/run_linux_fallback.sh
```

### 期待結果

- cargo run 成功
- GUI 起動
- libEGL/MESA/ZINK error 回避

# 2. renderer

## TC-P0-1D-002

### 目的

fallback 環境で egui renderer が正常動作するか確認する。

### 手順

1. GUI 起動
2. Panel 表示確認
3. 日本語表示確認

### 期待結果

- Panel 表示
- 日本語表示

# 3. WebView 影響

## TC-P0-1D-003

### 目的

fallback が WebView に影響しないか確認する。

### 手順

1. P0-2 実行
2. WebView 表示確認
3. 日本語確認

### 期待結果

- WebView 表示
- WebKitGTK 起動
- fallback 環境で安定動作
