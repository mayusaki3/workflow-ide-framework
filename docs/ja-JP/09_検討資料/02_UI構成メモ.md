# UI 構成メモ

## UI構造

```text
Page
  ↓
Tab
  ↓
Frame
  ↓
Scene
```

## Page

UI機能単位。

例：

- Editor
- MuJoCo View
- VRM View
- Runtime Status
- Build Log

## Frame

Page または Tab を表示する領域。

Dock 型 UI を想定する。

## Scene

用途別の UI 状態。

例：

- Simulation Scene
- Debug Scene
- AI Scene
- Asset Scene

## 自動保存

以下の状態を自動保存可能とする。

- Layout
- Tab 状態
- 最大化状態
- Scene 状態

## GPU Viewport

GPU 描画 Page を埋め込み表示可能とする。

## Command System

Menu / Context Menu / Toolbar / Shortcut を Command として統一管理する。
