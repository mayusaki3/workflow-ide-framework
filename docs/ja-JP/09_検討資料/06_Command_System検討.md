# Command System 検討

## 概要

UI 操作を Command として統一管理する。

## 対象

- Menu
- Context Menu
- Toolbar
- Shortcut
- Command Palette

## Context

Command は Context に応じて切り替え可能とする。

例：

- 選択状態
- Runtime 状態
- Page 種類
- Scene 種類

## 状態

Command は以下を持てる。

- enabled
- disabled
- checked
- hidden

## 目的

- UI 操作統一
- Shortcut 統一
- Context Menu 統一
- Toolbar 統一
