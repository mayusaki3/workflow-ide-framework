# 目的

Workflow IDE Framework は IDE 機能を持ったアプリケーション開発を容易にすることを目的としており、以下のことを実現できるようにする。

## 1. Workflow開発を統合するため

AI・外部ツール・Runtime・アプリケーションを、
共通のWorkflowとして扱えるようにする。

## 2. Runtime IDEを実現するため

単なるエディタではなく、
「実行中のWorkflow」を扱える開発環境を実現する。

## 3. AIと人間の協調開発を実現するため

AI Agent・LLM・人間・外部Runtimeを、
同じWorkflow上で協調動作できるようにする。

## 4. 再利用可能なWorkflow基盤を作るため

特定用途専用ではなく、
様々なシステムで利用可能なFrameworkを構築する。

## 5. 実行環境差異を吸収するため

ローカル・サーバー・ブラウザ・外部Runtimeなど、
異なる実行環境でも同じWorkflowを扱えるようにする。
