# 機能要件

Workflow IDE Framework は以下の機能を提供する。

## 1. Workflow 定義

利用者は Workflow を定義・保存・再利用できること。

## 2. Workflow 実行

Workflow を実行し、状態・結果・実行ログを確認できること。

## 3. Runtime 管理

Runtime の起動・停止・状態確認を行えること。

## 4. Runtime 監視

実行中 Workflow や Runtime 状態を監視可能であること。

## 5. API 提供

外部システムから Workflow や Runtime を操作可能な API 提供を考慮すること。

## 6. Command 統一管理

UI・API・外部Runtimeなど複数操作経路から、
同一操作を実行可能な構造を考慮すること。

## 7. Plugin 機構

Node・Runtime・UI・外部連携などを Plugin により追加可能であること。

## 8. GPU Viewport

MuJoCo や VRM preview などを表示可能な GPU Viewport を考慮すること。

## 9. 外部Runtime 連携

外部プロセス・外部Runtime・外部機器との連携を可能とすること。

## 10. AI 連携

LLM や AI Agent を Workflow 上で利用可能であること。

## 11. Workflow 保存

Workflow を保存・読み込み・共有可能であること。

## 12. 状態同期

複数 Runtime・UI・外部システム間で状態同期可能であること。
