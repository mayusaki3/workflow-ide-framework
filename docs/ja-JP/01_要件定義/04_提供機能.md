# 提供機能

Workflow IDE Framework は、以下の機能を利用者に提供する。  
これらを組み合わせてパッケージ化することで、専用 IDE 機能をアプリケーション利用者に提供することができる。

## 1. Workflowを作成できる

Nodeを組み合わせ、
処理の流れを構築できる。

## 2. Workflowを実行できる

作成したWorkflowを実行し、
結果や状態を確認できる。

## 3. Workflowを再利用できる

Workflowを保存・共有・再利用できる。

## 4. 外部システムと連携できる

外部Runtimeやサービスと接続できる。

## 5. 実行中Workflowを操作できる

一時停止・再開・強制停止・デバッグなどを行える。

## 6. AI支援を利用できる

AIによるWorkflow生成・補助・解析を利用できる。

## 7. 機能を拡張できる

Pluginや追加Nodeによって機能拡張できる。
