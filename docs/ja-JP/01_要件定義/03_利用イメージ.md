# 利用イメージ

## Discord Bot Workflow

「入力 → AI解析 → 条件分岐 → Discord返信」
のようなBot処理をWorkflowとして構築する。

## AI生成Workflow

「入力 → Prompt生成 → AI画像生成 → 保存」
のようなAI処理パイプラインを構築する。

## Runtime IDE

実行中Workflowの状態を確認し、
停止・再開・デバッグを行う。

## ロボット制御Workflow

外部Runtimeやマイコンと接続し、
ロボット制御フローを構築する。

## VR・シミュレーション連携

VRシステムやシミュレーション環境と接続し、
状態同期や制御連携を行う。

## 外部サービス連携

GitHub・OpenAI API・Discordなどの外部サービスと連携する。
