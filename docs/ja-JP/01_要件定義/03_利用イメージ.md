# 利用イメージ

Workflow IDE Framework を利用することで、どのようなことが可能になるか、例を示す。  
ここで示した例は、Workflow IDE Framework 単体の機能ではなく、利用者が開発した機能を組み合わせて実現されるものである。

## 1. Discord Bot Workflow

「入力 → AI解析 → 条件分岐 → Discord返信」
のようなBot処理をWorkflowとして構築する。

## 2. AI生成Workflow

「入力 → Prompt生成 → AI画像生成 → 保存」
のようなAI処理パイプラインを構築する。

## 3. Runtime IDE

実行中Workflowの状態を確認し、
停止・再開・デバッグを行う。

## 4. ロボット制御Workflow

外部Runtimeやマイコンと接続し、
ロボット制御フローを構築する。

## 5. VR・シミュレーション連携

VRシステムやシミュレーション環境と接続し、
状態同期や制御連携を行う。

## 6. 外部サービス連携

GitHub・OpenAI API・Discordなどの外部サービスと連携する。
