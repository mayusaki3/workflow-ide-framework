# Workflow アーキテクチャー

# 1. 概要

Workflow IDE Framework は Node Graph ベースの Workflow モデルを採用する。

# 2. Workflow 構造

Workflow は Node・Port・Connection・Graph により構成する。

# 3. Runtime 非依存

Workflow 定義は Runtime 非依存とする。

# 4. Event / State

Workflow 実行状態は Event / State モデルで扱う。

# 5. 今後の詳細仕様

- Node
- Port
- Connection
- Graph
- Execution Model
- Serialization
